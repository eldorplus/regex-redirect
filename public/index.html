URL Pattern Options

  • [protocol] - Protocol (ie. 'http://')
  • [subdomain] - Subdomain (ie. 'mail.', 'www.')
  • [domain] - Domain name (ie. 'google')
  • [tld] - Top level domain (ie. '.com', '.ca')
  • [hostname] - Hostname (ie. 'mail.google.ca')
  • [host] - Hostname + Port (ie. 'mail.google.ca:8080')
  • [pathname] - Path (ie. '/about/me')
  • [port] - Port (ie. ':8080')
  • [searchParams] - Query parameters (ie. '?user=bob')
  • [username] - Username (ie. 'thomas')
